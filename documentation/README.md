# AppToHTML Documentation

This folder documents the main modules in the application and provides editable
Draw.io diagrams for the current architecture.

## Contents

- `modules.md`
  Overview of the application modules, responsibilities, and boundaries.
- `crawler-module.md`
  Detailed explanation of the crawler and accessibility-driven capture flow.
- `data-and-state.md`
  Notes on persisted data, runtime state, and diagnostics.
- `architecture-overview.drawio`
  High-level module and dependency diagram.
- `crawler-sequence.drawio`
  Sequence-style view of the crawler lifecycle.

## Editing diagrams

The `.drawio` files are plain XML files intended to be opened with Draw.io /
diagrams.net. They are kept in source control so the diagrams evolve with the
codebase.
